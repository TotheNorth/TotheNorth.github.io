## 介绍

Antd 的[穿梭框](https://4x.ant.design/components/transfer-cn/)只提供 list to list 和 tree to list 两种模式，这里提供一种穿梭框可以支持 tree to tree 或 tree to list

| 参数       | 描述                                                |
| ---------- | --------------------------------------------------- |
| treeData   | 采集树数据源                                        |
| type       | 决定是 tree to tree（tree） 或 tree to list（list） |
| targetKeys | 已选条目                                            |
| disabled   | 是否禁用                                            |
| readOnly   | 是否只读                                            |
| onChange   | 数据变更回调                                        |
