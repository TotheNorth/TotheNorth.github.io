| onCancel | 模态窗关闭后的回调函数                                                                                                                                          |
| data     | 包含 url（视频资源地址）以及其他 video 标签属性的对象，详细展示逻辑见 [Video 组件](https://github.com/Hokkaii/agul-ui/blob/main/src/components/Video/README.md) |
