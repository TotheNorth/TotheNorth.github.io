### 介绍

| 参数      | 描述                                                                            |
| --------- | ------------------------------------------------------------------------------- |
| url       | 资源远程服务器地址                                                              |
| path      | 资源路径                                                                        |
| method    | 请求方式                                                                        |
| params    | 请求参数                                                                        |
| getOption | 参数是请求到的静态资源，在此方法中对数据进行整形而后返回 echarts 需要的配置数据 |
| option    | 静态的 echarts 配置数据                                                         |
